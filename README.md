# BarcodeToCloud Examples

This repository contains examples of how to use the BarcodeToCloud mobile app. BarcodeToCloud is a mobile app that allows you to scan barcodes and QR codes and execute actions based on the data scanned.

* ➡️ [Get from Android Play Store](https://play.google.com/store/apps/details?id=com.albertdev.barcodetocloud)
* ➡️ [Get from iOS App store](https://apps.apple.com/us/app/barcode-to-cloud/id1614121432)

## Examples

### Receiving goods in a warehouse

Receive items against a Purchase Order and increase available stock. For small shops with a couple of warehouse users, this model is recommended.
[Read More](Receive-goods-in-warehouse/README.md)

### Receiving goods in a warehouse (Advanced)

Receive items against a Purchase Order and increase available stock with a transaction/ledger model. For high volume warehouses with multiple users, this model is recommended.
[Read More](Receive-goods-in-warehouse-advanced/README.md)
